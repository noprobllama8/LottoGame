# Lotto Game
A simple lotto game developed for Computer networks course.

# Installation
To build project:

```sh
make
```
To execute server:

```sh
./server <server_port>
```

To execute clients:

```sh
./client <ip_server> <server_port>
```
To clean project:

```sh
make clean
```

If you are running server and clients in local choose ip: 127.0.0.1
